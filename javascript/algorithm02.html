2. 사용하지 않는 가장 작은 숫자 찾기

Q : 0부터 시작하는 양수(+)인 ID가 있습니다. 여러 ID를 보고 사용하지 않는 가장 작은 숫자를 찾는 함수를 작성해주세요.

입력 예제

  • [0,1,2,4] ⇒ 3
  • [0,1,2,3,4,5,6] ⇒ 7

주의할점

  • 정렬되지 않는 배열을 넣어도 동작해야 함
  • 중복된 숫자가 있는 경우도 동작해야 함

코드

const t2 = (ids) => {
    var cnt = 0
    while (true) {
        if (!ids.includes(cnt)) return cnt;
        cnt++;
        if (cnt==ids.length) break;
    }
    return cnt;
}

풀이

cnt라는 변수에 0부터 1씩 더해가며 배열의 내장 함수인 includes를 사용해서 존재하지 않는 수를 만났을 때 cnt를 리턴하게 했습니다. 여기서 한 가지 주의해야 할 점은 배열이 중간에 빠진 수 없이 쭉 이어질 때의 예외 케이스입니다. 이 예외 케이스는 cnt가 배열의 length와 같아질 때까지 존재하지 않는 숫자가 없었다면 그 때의 cnt에 +1 한 값을 리턴하게 했습니다.

6. 중간 숫자 찾기

Q : 3개의 숫자 중에 2번째 크기의 숫자 위치를 찾는 함수를 작성해주세요!

예시

  • [0,1,2] ⇒ 1
  • [5,1,2] ⇒ 2

코드

const t6 = (sample) => {
    return sample.sort(function (a, b) {
        return a - b;
    })[1];
}

풀이

주어진 배열을 숫자 크기순으로 정렬하기 위해 sort 함수에 a-b를 리턴하는 콜백함수를 인자로 넣어 숫자 크기순의 오름차순으로 정렬하였습니다. 이후 중간 값인 1번 인덱스의 값을 리턴하였습니다.

3. 승점구하기

Q : 점수를 보고 이기면 3점, 비기면 1점, 지면 0점으로 계산하여 전체 승점을 구하는 함수를 작성해주세요.

입력 예제

  • ["3:1", "2:2", "1:3"] ⇒ 4

주의할점

  • 10개의 경기 점수를 입력합니다.
  • 0점에서 최대 4점까지 입력합니다.

코드

const t3 = (games) => {
    let score = 0;
    for (let value of games) {
        if (value[0] > value[2]) score += 3;
        else if (value[0] < value[2]) score++;
    }
    return score;
}

풀이

경기 내용들이 배열 안에 string형태로 저장되어 있기 때문에 for of문으로 배열 안의 모든 요소들을 탐색하며 0번 인덱스 (내 점수) 와 2번 인덱스(상대의 점수)를 비교해서 score 변수에 승점을 누적해주었습니다.

5. 모음 찾기

Q : a, e, i, o, u가 몇개 들어 있는지 출력하는 함수를 작성해주세요!

예시

  • abracadabra ⇒ 5

코드

const t5 = (str) => {
    let count = 0
    const vowels = ["a", "e", "i", "o", "u"]
    for (let i of str)
        if (vowels.includes(i)) count++;
    return count;
}

풀이

a, e, i, o, u가 담긴 const 형태의 배열을 하나 만든 후 string의 자료형을 가진 str에서 for of 문을 이용하여 문자가 모음들이 담긴 배열에 있는지, 즉, 모음인지 비교하여 카운트해주었습니다.

7. 친구 찾기

Q : 이름이 4글자면 친구입니다. 친구를 찾는 함수를 작성해주세요!

예시

  • ["Ryan", "Kieran", "Mark"] => ["Ryan", "Mark"]

주의할점

  • 입력받은 순서를 유지해서 출력해주세요.

코드

const t7 = (arr) => {
    let answer = Array();
    for (let i of arr) if (i.length == 4) answer.push(i);
    return answer;
}}

풀이

주어진 배열의 원소들을 for of문으로 각각 접근하여 length 객체를 통해 길이가 4인지 비교한 후 4라면 answer 배열에 push로 후방삽입하여 리턴해주었습니다.

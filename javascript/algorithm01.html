1. 분기 숫자 구하기

Q : 1~12월중에 숫자를 입력받고 1분기, 2분기, 3분기, 4분기를 리턴하는 함수를 작성해주세요.

입력 예제

  • 3 ⇒ 1
  • 8 ⇒ 3
  • 11 ⇒ 4

주의할점

1부터 12까지의 숫자만 입력합니다.

코드

const t1 = (month) => {
    return parseInt((month + 2) / 3);
}}

풀이

month를 입력받았을 때 해당 분기를 출력해야 합니다. 분기는 3개월 단위로 끊어지므로 3으로 나눈 몫을 출력하면 됩니다. 여기서 주의할 점은 1, 2, 3을 3으로 나눈 몫은 0, 0, 1 이란 점입니다. 따라서 month에 값을 더해 처리해주어야 하는데 조건은 다음과 같습니다.

1, 2, 3월은 1분기 -> 3으로 나눈 몫이 1이 되려면 [3, 4, 5] 로 값을 변경시켜야 함.

4, 5, 6월은 2분기 -> 3으로 나눈 몫이 2가 되려면 [6, 7, 8] 로 값을 변경시켜야 함.

...

따라서 month에 2를 더해서 3으로 나눈 몫을 분기로 출력하면 됩니다. 이 때, 주의할 점은 JS의 나누기 연산자의 값은 float형이므로 parseInt()함수를 통해 정수로 변환해주어야 합니다.
